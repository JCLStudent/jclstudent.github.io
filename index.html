Hi, so you want to make a webpage huh?

First go to https://www.barebones.com/products/bbedit/download.html to download a simple source code editor.

Click the download button next to the text "Disk Image"


You will see a small box on the bottom left-hand side of the window. Double click on it.


Wait a minute and this window will appear.


-----------------------------------


Now that you have a source code editor download the Github client.

Go to https://desktop.github.com/ and click the macOS link.
